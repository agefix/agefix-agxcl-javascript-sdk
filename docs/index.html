‚ö° Quick Start

import { AgefixClient, TokenContract } from '@agefix/agxcl-sdk';

// Initialize client
const client = new AgefixClient({
  rpcUrl: 'https://rpc.agefix.com',
  chainId: 'agefix-mainnet-1',
  privateKey: 'your-private-key'
});

// Deploy a token contract
const token = new TokenContract(client);
const deployment = await token.deploy(
  'MyToken',
  'MTK',
  '1000000'
);

console.log('Contract deployed at:', deployment.contractAddress);

// Transfer tokens
const tx = await token.transfer(
  '0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb',
  '100'
);
console.log('Transfer completed:', tx.txHash);

üìö API Reference

AgefixClient

Main client for interacting with the AgeFix blockchain.

constructor(config: AgefixConfig)

Creates a new client instance.

Parameters:

  • config.rpcUrl - RPC endpoint URL
  • config.chainId - Chain identifier
  • config.privateKey - Private key for signing transactions (optional)
deployContract(contractCode: string, constructorArgs?: any[])

Deploys a new AGXCL smart contract to the blockchain.

Parameters:

  • contractCode - AGXCL contract source code
  • constructorArgs - Array of constructor arguments

Returns: Promise<ContractDeployment>

queryContract(contractAddress: string, method: string, args?: any[])

Executes a read-only contract query.

Parameters:

  • contractAddress - Deployed contract address
  • method - Method name to call
  • args - Method arguments

Returns: Promise<QueryResult>

executeTransaction(contractAddress: string, method: string, args?: any[], value?: string)

Executes a state-changing contract transaction.

Parameters:

  • contractAddress - Deployed contract address
  • method - Method name to call
  • args - Method arguments
  • value - Amount of AGX tokens to send (optional)

Returns: Promise<TransactionResult>

getBalance(address: string)

Gets the AGX token balance for an address.

Parameters:

  • address - Account address to query

Returns: Promise<string>

estimateGas(contractAddress: string, method: string, args?: any[])

Estimates gas cost for a transaction.

Parameters:

  • contractAddress - Contract address
  • method - Method name
  • args - Method arguments

Returns: Promise<number>

TokenContract

Helper class for creating and managing token contracts.

deploy(name: string, symbol: string, totalSupply: string)

Deploys a new token contract with specified parameters.

Parameters:

  • name - Token name (e.g., "MyToken")
  • symbol - Token symbol (e.g., "MTK")
  • totalSupply - Initial token supply

Returns: Promise<ContractDeployment>

balanceOf(address: string)

Gets token balance for an address.

transfer(to: string, amount: string)

Transfers tokens to another address.

approve(spender: string, amount: string)

Approves another address to spend tokens.

NFTContract

Helper class for creating and managing NFT contracts.

deploy(name: string, symbol: string)

Deploys a new NFT contract.

mint(to: string, uri: string)

Mints a new NFT with metadata URI.

ownerOf(tokenId: number)

Gets the owner of an NFT.

Custom Contract Interaction

// Deploy custom contract
const deployment = await client.deployContract(`
contract MyContract {
  state {
    uint256 value;
  }
  
  function setValue(uint256 newValue) public {
    value = newValue;
  }
  
  function getValue() public view returns (uint256) {
    return value;
  }
}`);

// Execute transaction
await client.executeTransaction(
  deployment.contractAddress,
  'setValue',
  [42]
);

// Query state
const result = await client.queryContract(
  deployment.contractAddress,
  'getValue'
);
console.log('Value:', result.data);
